# Woven‑Style Frontend Assessment

This repo contains a take‑home assessment modeled after real mid‑senior frontend interviews.

📄 **Read the full prompt:** `ASSESSMENT.md`

## Quick Start
```zsh
npm install
npm run dev
```

## Scripts
```zsh
npm run build
npm run lint
```

## Notes for Candidates
- Use **React + TypeScript** only.
- No backend — mock all data locally.
- Prefer clarity and UX polish over extra features.
- Document tradeoffs and improvements in this `README.md` when you submit.
